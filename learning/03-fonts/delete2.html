**font** : A collection of letter shapes or `glyphs` that share a common design. **typeface** : A set of one or more fonts that share a common design. **unicode** : A standard for encoding every character in every language in the world. **codepoint** : The numerical identifier used to represent a character in Unicode. **web-font** : A font that is loaded from the web, not your computer. **`@font-face`** : A `CSS` mechanism for loading web fonts. **truetype** : A font format which is widely supported on the web. **opentype** : A new font technology that is widely supported in browsers and which supports many advanced typographical features.
